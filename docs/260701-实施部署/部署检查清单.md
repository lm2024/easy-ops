# EasyOps 部署检查清单

> 逐项打勾确认，确保不遗漏
> 无 Nginx，前端使用 Vite dev server

---

## 编译阶段

- [ ] 服务器已安装 JDK 8 (`java -version` 输出 1.8.x)
- [ ] 服务器已安装 Maven 3.6+ (`mvn -version`)
- [ ] 服务器已安装 Node.js 18+ (`node -v`)
- [ ] 代码已传输到服务器 `/opt/easy-ops/`
- [ ] 后端编译成功：`cd /opt/easy-ops/backend && mvn clean package -DskipTests`
- [ ] Server JAR 存在：`ls server/target/ops-platform-server-1.0.0-SNAPSHOT.jar`
- [ ] Agent JAR 存在：`ls agent/target/ops-platform-agent-1.0.0-SNAPSHOT.jar`

## Server 配置修改

- [ ] `server/.../application.yml` 中 `server.path` 改为绝对路径
- [ ] `server/.../application.yml` 中 `spring.datasource.url` 改为绝对路径
- [ ] `server/.../application.yml` 中 `cors.allowed-origins` 添加生产 IP:8080
- [ ] `server/.../application.yml` 中 `logging.level` 生产环境改为 INFO
- [ ] 环境变量 `JWT_SECRET` 已设置（≥32 字符）

## Agent 配置修改

- [ ] `agent/.../application.yml` 中 `agent.server-url` 指向 Server IP:8081
- [ ] 环境变量 `AGENT_TOKEN` 已设置（与 Server 注册节点一致）
- [ ] 环境变量 `AGENT_NODE_NAME` 已设置

## 前端配置修改

- [ ] `frontend/vite.config.ts` 中 `server.port` 改为 `8080`
- [ ] `frontend/vite.config.ts` 中 `server.host` 设置为 `'0.0.0.0'`
- [ ] `frontend/vite.config.ts` 中 proxy 配置正确（`/api` → `:8081`, `/ws` → `ws://:8081`）
- [ ] 前端依赖安装成功：`cd /opt/easy-ops/frontend && npm install`

## 防火墙

- [ ] 端口 8080 已开放（前端 Vite dev server）
